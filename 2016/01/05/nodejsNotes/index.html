nodejs笔记

发表于 2016-01-05 |

学习地址

工具

node的版本管理工具:NVM(Node Version Manager)(https://github.com/creationix/nvm)

node包管理工具:npm(node package manage)(https://github.com/npm/npm)

npm修改淘宝源: 添加代码registry = https://registry.npm.taobao.org到~/.npmrc

npm init: 初始化package.json
npm install --save: 安装包并将包版本名称写入package.json
--save-dev: 安装可执行的包在本地

包

express: web框架

引入依赖 var express = require('express');
require对应与express中的module.exports or exports(两者之间的区别),此时得到的express并不是实例对象,个人理解为是一段可执行的code。
var app = express();

此时的app才是一个express实例

superagent: 类Ajax组件

cheerio: 类jquery组件

eventproxy: 事件异步处理

asyac: 控制并发的工具

mocha:测试框架

should: 断言库

chai: 全栈断言库

istanbul: 测试率覆盖工具

nodemon:Node的监测文件改动并自动编译

phantomjs: 浏览器测试工具

supertest: 所有兼容 connect 的 web 框架进行集成测试(配合express)
